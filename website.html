How the Riddle Works

The riddle follows a simple algebraic sequence that always results in the same number, regardless of your starting number. Let's see the steps:

  1. 1

    Choose any number

  2. 2

    Double it (Multiply by 2)

  3. 3

    Add 6

  4. 4

    Divide it by 2

  5. 5

    Subtract the number you started with

Why does this always work?

Algebraically, if your starting number is x:
((2x + 6) / 2) - x = (2x/2 + 6/2) - x = (x + 3) - x = 3

After these steps, you'll always end up with 3 no matter what number you started with!

Programming Concepts

This simple riddle project in Java demonstrates several fundamental programming concepts:

// MathRiddle.java - Demonstrates basic programming concepts
public class MathRiddle {
    public static void main(String[] args) {
        // Step 1: Choose any number (variable declaration)
        int originalNumber = 5;
        
        // Step 2: Double it (arithmetic operation)
        int doubledNumber = originalNumber * 2;
        
        // Step 3: Add 6 (compound assignment)
        doubledNumber += 6;
        
        // Step 4: Divide by 2 (type conversion)
        double halfNumber = doubledNumber / 2.0;
        
        // Step 5: Subtract original number
        double result = halfNumber - originalNumber;
        
        // Output the result
        System.out.println(result); // Always 3.0!
    }
}
